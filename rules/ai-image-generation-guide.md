## Best Practices

### Prompt Engineering:
- Be specific about style and mood
- Include technical specifications (resolution, format)
- Reference existing art styles or artists
- Iterate and refine prompts based on results

### Integration Workflow:
1. Generate base images with AI tools
2. Post-process in design software (Photoshop, Figma)
3. Optimize for web (compress, convert formats)
4. Implement in prototypes with CSS/Canvas/WebGL

### Legal Considerations:
- Check licensing terms for each tool
- Ensure commercial use rights
- Consider attribution requirements
- Use Adobe Firefly for commercial projects
